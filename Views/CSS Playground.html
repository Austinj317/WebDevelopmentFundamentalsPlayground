CSS Items

Uses

  • Gives the apperance of your site, a unique look and feel.
  • Alters colors, fonts, backgrounds, etc.
  • Implements effects and animations, although its not as powerfull as jQuery/JavaScript

CSS Statements

Four main components of CSS statement

  1. Declaration: Whole CSS statements in curly braces.
  2. Selector: Name of the part you want to style.
  3. Property: Properties to edit (e.g., width, height, background,etc.)
  4. Value: Numbers, colors, URLs - things to give value to the property.
